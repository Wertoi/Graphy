Part Name

String value which defines the name of the part in windows explorer without extension.
Example: C:\...\Part1.CATPart. Write "Part1".

Marking Name

String value which defines the name of the marking.

Is text

Boolean value which defines if the marking is a text or an icon/image.
- 0: for text.
- 1: for icon/image.

Text

String value which defines the text to write.
Can be null if IsText = 1.

Is bold, Is italic, Is underline, Is strike through

Boolean value which defines boldness, italic and text line decorations for text marking.
- 0: for false.
- 1: for true.
Can be null if IsText = 1.

Font family

String value which defines the name of the font family for text marking.
The font family must be installed in windows.
Can be null is IsText = 1.

Icon path data

String value which defines the icon/image to draw in xaml format.
Can be null is IsText = 0.

Marking height

Number which defines the height of the marking in the current Catia length unit.
Value must be strictly superior to 0.

Extrusion height

Number which defines the height of the extrusion in the current Catia length unit.
- Negative value for substracting the marking body of the main body in the +Z direction of the reference axis system.
- Positive value for adding the marking body to the main body in the -Z direction of the reference axis system.

Horizontal alignment

Integer value which defines the horizontal alignment of the marking compared to the reference point and the X axis direction.
- 0: for left alignment.
- 1: for center alignment.
- 2: for right alignment.

Vertical alignment

Integer value which defines the vertical alignment of the marking compared to the tracking curve.
- 0: for top alignment. Marking is below tracking curve.
- 1: for center alignment. Marking is centered on the tracking curve.
- 2: for bottom alignment. Marking is above tracking curve.

Projection surface name

String value which defines the name of the surface in Catia used to project the marking.
- If the projection surface is composed by different sub-surfaces, all sub-surfaces must have the same natural normal.

Tracking curve name

String value which defines the name of the tracking curve in Catia used to guide the marking.
- The tracking curve must be on the projection surface.
- If the tracking curve is composed by different sub-curves, all sub-curves must have the same natural direction.

Reference point name

String value which defines the name of the reference point in Catia used to define the starting of the marking.
- The reference point must be on the tracking curve.

Axis system name

String value which defines the name of the axis system in Catia used to define the directions of the marking.
- The axis system origin must be the reference point.
- The X axis must be the tangent to the tracking curve passing by the reference point. Its direction points the marking direction (left to right for example).
- The Y axis is the result of X and Z axis. Its direction defines the top and the bottom.
- The Z axis must be the normal of the projection surface passing by the reference point. Its direction points the matter adding direction.
